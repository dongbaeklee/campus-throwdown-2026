# 캠퍼스 스로우다운 홍보 에이전트 팀 하네스

이 폴더는 `캠퍼스 스로우다운 2026` 참가 모집과 홍보 실행을 위한 로컬 에이전트 팀 하네스입니다. 외부 패키지 없이 Node.js만 있으면 실행됩니다.

## 구성

- `team.json`: 프로젝트 사실 정보, 에이전트 목록, 실행 순서
- `prompts/`: 에이전트별 역할 프롬프트
- `harness.mjs`: 팀 정의와 프롬프트를 묶어 실행 브리프와 초안 산출물을 생성하는 러너
- `runs/`: 실행 결과가 저장되는 폴더

## 에이전트 팀

1. 기획 리드: 목표, 의사결정, 마일스톤, 부서별 요청사항 정리
2. 마케팅 리드: 메시지, 채널, 콘텐츠 브리프, 성과 지표 조율
3. 운영 리드: 신청, 문의, 현장 운영, 리스크 체크리스트 작성
4. 재무 리드: 참가비 수익, 예산 항목, 손익 리스크, 정산 기준 점검
5. 캠페인 전략가: 홍보 각도와 2주 실행 플랜 설계
6. 카피라이터: SNS, 포스터, CTA 문구 작성
7. 채널 플래너: 인스타그램, 커뮤니티, 단톡, 포스터 운영 계획
8. 제휴 담당자: 동아리, 학생회, 박스, 브랜드 제휴 메시지 작성
9. 전환 분석가: 신청 페이지 전환율 개선안 제안
10. 운영 검수자: 사실 정보와 운영 리스크 검토

## 실행

```bash
node agents/harness.mjs "인스타그램과 대학 커뮤니티 중심으로 참가 신청을 늘리는 캠페인"
```

결과는 아래 두 위치에 저장됩니다.

- `agents/runs/latest.md`
- `agents/runs/<실행시각>.md`

## 운영 방식

1. `team.json`의 사실 정보가 실제 운영 정보와 맞는지 먼저 확인합니다.
2. 실행 목표를 한 문장으로 넣어 하네스를 실행합니다.
3. `runs/latest.md`의 초안을 검토합니다.
4. 실제 LLM이나 자동화 도구에 붙일 때는 `에이전트 지시문` 섹션을 에이전트별 시스템/개발자 프롬프트로 사용합니다.
5. 게시 전에는 반드시 운영 검수자의 체크리스트를 통과시킵니다.
